Installation

Here's how to install this blog template on your website:


                        # Clone the repository
                        git clone https://github.com/yourusername/technotes.git

                        # Navigate to the project folder
                        cd technotes

                        # Install dependencies
                        npm install

                        # Start development server
                        npm run dev

We'll be using yaml format for all examples below as it is recommended over toml since it's easier to read.

You can find YAML to TOML converters if needed.
